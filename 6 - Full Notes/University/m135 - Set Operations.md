# m135 - Set Operations

**Union:** Set of all elements belonging to either $S$ or $T$ (or both) 
- $S\cup T=\{x:x\in S \text{ OR } x\in T\}$ 

**Intersection:** Set of all elements belonging to both $S$ and $T$ 
- $S\cap T=\{x:x\in S \text{ AND } x\in T\}$ 

**Set difference:** Set of all elements belonging to $S$ but not $T$ 
- $S-T=S\textbackslash T=\{x:x\in S \text{ AND } x\notin T\}=\{x:(x\in S)\land (\neg(x\in T))\}$   

**Complement:** Set of all elements not in $S$ 
- AKA the complement of a set $S$ whose elements all belong to $U$ is all elements in $U$ but not $S$ 
- $\overline{S}=\{x:x\notin S\}$ 
- $\overline{S}=\{x\in U:x\notin S\}=\{x\in U: \neg(x\in S)\}$
