# IMPORTANT - Proof Notation

**Terminology:** 
- *Image of* → output when you plug in x to a function (f(x))
- *Surjective (onto)* → every element in the output set is hit
- *Injective (one to one)* → there is only one input for every output
- *Bijective* → Surjective + Injective
- $f: X \rightarrow Y$:
	- f is a function that takes an input from set X and gives an output in set Y
	- X → domain
	- Y → range
