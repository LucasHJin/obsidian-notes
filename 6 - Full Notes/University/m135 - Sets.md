# m135 - Sets

**Set:** a well-defined, unordered collection of distinct objects (use uppercase letters)
- Each object in this collection → **element/member** of the set (use lowercase letters)
- $\in$ → part of set
- $\notin$ → not part of set
- $|$ → such that
