# cs145 - DrRacket

**DrRacket:** an IDE to write Racket in
- *Top window* → definitions window to write/save programs
- *Bottom window* → interactions window to evaluate expressions
	- Red italics → something has gone wrong
- **Commands:** 
	- Ctrl R Resets
	- Can stop with big red square
	- Can go back to previous expressions
- **Tweaking DR:** 
	- Change language to change details → i.e. style of constant, fractions, etc.
	- *Auto indent* → select everything and hit tab
		- Indents nicely for you
