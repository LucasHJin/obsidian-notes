# m135 - Proving If and Only If Statements

Relates to → [[m135 - If and Only If]] 

**Proof method:** 
- To prove the Iff statement “$A\iff B$”
	- Prove the following implications → $A\implies B$ and $B\implies A$ 
- To prove the UQS, “$\forall x\in S, P(x)\iff Q(x)$”, show either
	- $P(x)\implies Q(x)$ and $Q(x) \implies P(x)$ for an arbitrary value of $x$ or
	- Prove the following UQI → $\forall x\in S, P(x)\implies Q(x)$ and $\forall x\in S, Q(x)\implies P(x)$
