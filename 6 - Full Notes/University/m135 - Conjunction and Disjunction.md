# m135 - Conjunction and Disjunction

**Compound statement:** statement composed of several individual statements (called ***component statements***) 
- Combined with logical operators
	- Both below typically embedded in quantified statements
- **Conjunction:** $A\land B$ → A and B
	- Only true when they are both true (exclusive)
- **Disjunction:** $A\vee B$ → A or B
	- True if at least a single value is true (inclusive
