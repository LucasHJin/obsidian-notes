# Need to know shortcuts and formatting

1. Command B, I to bold and italicize
2. # with space to create titles, subtitles, etc.
3. ~~ ~~ for strikethrough
4. == == for highlight
5. # with **no space** to create tags
6. double brackets [[]] to link to other notes
7. [](()() to link to an external source
	1. If blank spaces in url, replace with %20
8. Above with ! at front for external images
	1. Add |AxB after the text within [] to change dimensions
9. > for a quote (or "")
	1. Each new line needs new >
	2. Add [!info] as first part of quote to create callout (like sticky note)
10. - [] for checklist
11. 2 back ticks for inline code blocks ` `` `
12. 6 back ticks for entire code blocks
	1. Can add language code for coloured highlighting
13. [^1] for footnotes
	1. [^1]: for referenced text
