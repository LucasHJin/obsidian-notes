# Correlation strength explained

**Correlation strength:** A measure of how close a set of data is clustered around its [[Linear correlation explained|least-squares regression line]].
- Measured with **Pearson product-moment correlation coefficient** ($r$):
	- $-1\leq r\leq 1$, indicates the strength and direction of a correlation.
		- If $r=1$ -> perfect positive linear correlation: all the data points lie on the line
		- If $r=-1$ -> perfect negative linear correlation: all the data points lie on the line
		- If $r=0$ -> no linear correlation: no best-fitting line can be drawn.
	- ==Closer to $r=\pm 1$ -> stronger linear correlation==

**Calculating Pearson's $r$:**
- ALWAYS CALCULATOR ON TEST
	- GDC stats calc
- $r=\dfrac{\sum{(x-\bar{x})(y-\bar{y})}}{\sqrt{\sum{(x-\bar{x})^2}\sum{(y-\bar{y})^2}}}=\dfrac{\sum{xy}-n\bar{x}\bar{y}}{\sqrt{(\sum{x^2}-n\bar{x}^2)}\sqrt{(\sum{y^2}-n\bar{y}^2)}}$
