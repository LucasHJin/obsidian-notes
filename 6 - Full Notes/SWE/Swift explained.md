# Swift explained
A modern, fast, and type-safe programming language developed by Apple to build IOS native apps.
- Built on top of Objective-C
- Strong choice for developing at Apple

**IMPORTANT:** https://developer.apple.com/swift-student-challenge/
- Student developers showcase their skills and creativity using Swift playground
- **Requirements:** 
	- A Swift Playgrounds project (.swiftpm) that runs in the latest version of the Swift Playgrounds app.
	- Must be under 3 minutes of interaction.
	- Can include code, graphics, sound, and more.
- **Example:** https://www.youtube.com/watch?v=1A_DqxcCGCM
