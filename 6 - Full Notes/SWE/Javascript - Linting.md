# Javascript - Linting

**Linter:** spellchecker for the code → checks for mistakes, bad practices, or formatting problems (and gives fixes) 
- Based on a set of [style rules](https://github.com/airbnb/javascript) 
- Helpful for formatting and not having to memorize style guides 

**ESLint:** Most popular linter for javascript
- Can auto install for React/Next project
- How to config → https://eslint.org/docs/latest/use/getting-started

**Formatters:** format the code based on a set of rules (only layout, not style errors)
- Most popular → **Prettier** 

*Note:* You can use ESLint and Prettier extensions on VSCode to see the problems/format the code without extra steps
