# cURL explained

**Client URL (cURL):**  command-line tool used to send [[HTTP explained|HTTP]] requests (and other types of requests) from your terminal
- Can call APIs, test endpoints, download files, send POST/GET/PUT requests, add headers, auth and data

Allows you to talk to [[REST explained|REST APIs]] 
- Can simulate any REST operation
- Good for testing + debugging
