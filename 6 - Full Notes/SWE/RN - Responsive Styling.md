# RN - Responsive Styling

| Use                              | Why                                       |
| -------------------------------- | ----------------------------------------- |
| **Default `number` units**       | Already device-independent (like `dp`)    |
| **`Dimensions.get()`**           | To use percentages of screen width/height |
| **Flexbox**                      | Auto-responsiveness across screen sizes   |
| **SafeAreaView**                 | Prevent overlap with notches              |
| **Scaling libraries** (optional) | Fine-tuned responsive sizing              |
