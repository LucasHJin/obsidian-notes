# RN - Expo

**What is it?**
- a framework and platform built on top of React Native that makes it easier and faster to build mobile apps

**Benefits:** 
- Don’t need to worry about native languages at all
- Can scan with Expo Go App
- Has built in APIs (don’t need to access native configs)
