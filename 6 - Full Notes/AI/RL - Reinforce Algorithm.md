# RL - Reinforce Algorithm

**Reinforce algorithm:** algorithm in the policy gradient methods (reinforce good actions)
- **Objective:** expected return (function) that we want to maximize
	- **Gradient** → points in direction of steepest increase (how much the function changes when the input changes) 
		- **Gradient descent** → to minimize the return of a function
		- **Gradient ascent** → to maximize the return of a function
	- If it’s a good gradient → encourage it, else discourage it
